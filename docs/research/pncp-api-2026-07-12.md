## Endpoint de publicação validado

- Endpoint: `GET /api/consulta/v1/contratacoes/publicacao`
- Path documentado no OpenAPI: `/v1/contratacoes/publicacao`
- Servidor documentado: `/api/consulta`

## Parâmetros confirmados

Obrigatórios:

- `dataInicial`
- `dataFinal`
- `codigoModalidadeContratacao`
- `pagina`

Opcionais:

- `codigoModoDisputa`
- `uf`
- `codigoMunicipioIbge`
- `cnpj`
- `codigoUnidadeAdministrativa`
- `idUsuario`
- `tamanhoPagina`

## Formato de data confirmado

- Formato aceito: `yyyyMMdd`
- Exemplo válido: `20260514`
- Exemplo rejeitado em chamada real: `2026-05-14` retornou HTTP `422`

## Paginação real confirmada

Resposta `200` contém:

- `data`
- `totalRegistros`
- `totalPaginas`
- `numeroPagina`
- `paginasRestantes`
- `empty`

Limite documentado:

- `tamanhoPagina`: mínimo `10`, máximo `50`

## Endpoints oficiais usados para enriquecimento

- Detalhe da contratação: `GET /api/consulta/v1/orgaos/{cnpj}/compras/{ano}/{sequencial}`
- Itens: `GET /api/pncp/v1/orgaos/{cnpj}/compras/{ano}/{sequencial}/itens`
- Documentos: `GET /api/pncp/v1/orgaos/{cnpj}/compras/{ano}/{sequencial}/arquivos`
- Órgão: `GET /api/pncp/v1/orgaos/{cnpj}`
- Unidades do órgão: `GET /api/pncp/v1/orgaos/{cnpj}/unidades`

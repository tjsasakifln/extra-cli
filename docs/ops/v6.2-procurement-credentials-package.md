## 1. SKU recomendado (Netcup preferencial)

1. Conta em https://www.netcup.eu (ou netcup.com)
2. Root Server classe **~RS 4000**: 32 GB RAM, CPU dedicada, ~1 TB NVMe
3. Localização: preferência US-East se disponível; senão DE com teste PNCP obrigatório
4. Add-on: backup snapshot / storage se oferecido
5. Registrar IPv4, console, dados fiscais

1. Conta https://console.hetzner.cloud

## 2. Credenciais a obter (checklist)

### Infra

| Credencial | Onde guardar | Usado por |
|------------|--------------|-----------|
| SSH host + user | `.env.production` / secrets vault | deploy, ops |
| SSH private key | agent local / 1Password | SSH |
| `sudo`/root policy | doc ops | admin |
| Firewall allowlist | provedor | SSH 22, deny all else |
| Backup remote (`user@storage`) | `BACKUP_STORAGE_BOX_SSH` | `scripts/backup-database.sh` |
| DNS A record (opcional) | DNS provider | HTTPS futuro |

### Aplicação / DB

| Credencial | Env | Notas |
|------------|-----|-------|
| `DATABASE_URL` / `LOCAL_DATALAKE_DSN` | prod | PostgreSQL 16 local na VPS |
| `POSTGRES_PASSWORD` | prod | forte, rotacionável |

### Fontes de dados

| Fonte | Env / secret | Obrigatório? |
|-------|--------------|--------------|
| **ciga_ckan (DOM canônico)** | **nenhum** | Não — público |

| pncp | nenhum (API pública) | — |
| pcp / compras_gov | conforme crawler | geralmente público |
| doe_sc | `DOE_SC_LOGIN`, `DOE_SC_PASSWORD` | se ativar |
| mides_bigquery | `GOOGLE_APPLICATION_CREDENTIALS` | se ativar |

# Truth Audit Checklist

## Purpose
Conservative DoD truth reconciliation. Any NO blocks READY claims.

## Items

- [ ] DOD.md hash recorded
- [ ] Supersession sections read (including §44 LOCAL_RESILIENCE_READY)
- [ ] Each evaluated item has classification enum
- [ ] Evidence is executable or explicitly human-validated
- [ ] Fixtures/mocks not used as live proof
- [ ] DB row presence not treated as coverage
- [ ] Open PR not treated as main
- [ ] Contested READY seals set to NOT_READY
- [ ] Allowed claims listed
- [ ] Forbidden claims listed
- [ ] Veto recorded if false green risk
- [ ] No DoD checkboxes modified in this audit
